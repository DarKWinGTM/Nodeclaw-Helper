NodeClaw Helper Scripts

Public helper distribution for NodeClaw

This Pages site is published from Nodeclaw-Helper/script/. It is the public distribution surface for the checked helper scripts.

Source authority still remains in NodeClaw-platform/script/. This publish subtree exists inside the main repo so the working basis stays in-platform.

Supported tools

claude-code codex openclaw opencode zed

Current support boundary

  • Shell helper paths can apply changes where the checked script supports it.
  • PowerShell helper paths remain scaffold-first and dry-run-only.
  • Do not assume hosted curl | bash install is officially live.
  • openclaw requires the local openclaw command to already exist in PATH.

Start here

bash ./launcher.sh list
NODECLAW_API_KEY="<nodeclaw_access_key>" bash ./launcher.sh dry-run --tool claude-code

Prefer --dry-run first. Inspect the script before running it. Pass secrets through environment variables.
